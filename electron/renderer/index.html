📖 Chrome CDP 调试模式设置指南

⚠️ 重要提示：

在启动 CDP 模式前，请完全关闭所有 Chrome 窗口！

步骤 1：关闭所有 Chrome

完全关闭所有 Google Chrome 浏览器窗口（包括后台进程）

步骤 2：启动 Chrome 调试模式

选择以下任一命令，在命令提示符（CMD）中运行：

方法 A - 简单命令：

chrome.exe --remote-debugging-port=9222

方法 B - 完整路径（推荐）：

"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222

方法 C - 32位 Chrome：

"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222

步骤 3：确认 Chrome 启动

Chrome 会启动并显示提示："Chrome 正在被自动化测试软件控制"

✅ 这表示 CDP 模式已成功启动

步骤 4：使用应用

1. 关闭此设置指南

2. 点击任意网站按钮（如 ChatGPT）

3. 应用会自动连接到已启动的 Chrome

🎯 首次使用 ChatGPT

  1. 按照上述步骤启动 CDP 模式
  2. 点击 "🤖 ChatGPT" 按钮
  3. 在打开的 Chrome 中手动登录 ChatGPT
  4. 登录状态会自动保存，下次无需重新登录
💡 提示：

一旦登录成功，您可以关闭 CDP Chrome，下次正常使用即可。

登录信息已保存在您的 Chrome 用户数据中。

❓ 常见问题

Q: 端口 9222 被占用怎么办？

A: 使用其他端口，如：--remote-debugging-port=9223

Q: 如何停止 CDP 模式？

A: 直接关闭 Chrome 窗口即可

Q: 每次都要启动 CDP 模式吗？

A: 不需要。登录一次后，可以正常使用 Chrome
